## 🚀 使用方法

### 创建文件响应接口
1. 点击"新增接口"
2. 填写基本信息（名称、方法、URL）
3. 在响应体编辑中选择"文件响应"
4. 上传要返回的文件
5. 保存接口

### 测试文件接口
```bash
# 测试文件下载接口
curl -X GET "http://localhost:8344/api/download" -o downloaded_file

# 查看响应头
curl -I "http://localhost:8344/api/download"
```
