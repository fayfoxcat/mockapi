# 文件上传功能测试

## 新增功能说明

### 1. 返回类型选择
- **JSON响应**: 传统的JSON数据返回
- **文件响应**: 返回文件流，支持各种文件类型

### 2. 文件上传功能
- 支持上传任意类型的文件
- 自动生成唯一文件名避免冲突
- 保存在 `data/uploads/` 目录下
- 自动识别文件类型和大小

### 3. 界面改进
- 响应类型选择器（单选按钮）
- 文件上传区域（拖拽上传）
- 文件信息预览
- 响应类型标识徽章

## 测试步骤

1. 启动服务器：`./target/release/mock-api-server`
2. 访问：http://localhost:8344
3. 创建新接口
4. 选择"文件响应"类型
5. 上传测试文件
6. 保存接口
7. 测试接口调用

## API端点

- `POST /api/upload` - 文件上传
- 其他现有端点保持不变

## 数据结构变更

MockApi 新增字段：
- `responseType`: "json" | "file"
- `fileName`: 原始文件名
- `filePath`: 存储路径
- `contentType`: MIME类型
